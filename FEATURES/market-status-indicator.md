# Feature: 市场状态指示器

## 需求背景

美股交易时间与北京时间有昼夜颠倒的时差，且美国有夏令时变动。需要一个实时市场状态指示器，在主页展示当前美股是否开盘，以及距离开盘/收盘的时间。

## 需求总结

| 项目 | 决策 |
|------|------|
| 时区支持 | 双时区（美东 EDT/EST + 北京时间 CST） |
| 夏令时 | 自动识别（3月第二个周日→11月第一个周日） |
| 状态 | 盘前 / 盘中 / 盘后 / 休市 |
| 前端展示 | 主页 header 状态指示器 |

---

## 功能边界

### 做什么
- 实时显示美股当前状态（盘前/盘中/盘后/休市）
- 双时区显示当前时间
- 自动处理夏令时切换
- 显示距离开盘/收盘的倒计时

### 不做什么
- 不处理节假日（仅处理周末）

---

## 核心设计

### API

```
GET /api/market/status
  → services/market_status.py: 计算当前美东时间 → 判断状态
  → 返回 {
      is_open, status,  // "pre-market" | "open" | "after-hours" | "closed"
      eastern_time, beijing_time,
      next_event, next_event_time, next_event_label
    }
```

### 交易时段
- 盘前：美东 04:00–09:30
- 盘中：美东 09:30–16:00
- 盘后：美东 16:00–20:00
- 休市：美东 20:00–04:00（次日）

### 夏令时规则
- 开始：3月第二个周日 02:00 → 03:00（EDT, UTC-4）
- 结束：11月第一个周日 02:00 → 01:00（EST, UTC-5）

---

## 边界条件

- 夏令时切换日当天状态正确过渡
- 周末永远显示休市
