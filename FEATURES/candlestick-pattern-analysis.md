# Feature: K线形态识别与分析

## 需求总结

| 项目 | 决策 |
|------|------|
| 触发方式 | 点击单根 K 线柱，仅当 "K线+量" 切换组激活时 |
| 检测范围 | 数据充足时检测多根+单根形态，不足时仅检测单根形态 |
| 展示方式 | 弹出窗口（modal） |
| 多形态命中 | 全部展示 |
| 展示内容 | 形态名称、方向、形状描述、原理、位置要求、可靠性星级、确认信号、实战注意事项 |
| 检测方式 | 纯前端 JavaScript 固定规则判断，不使用 AI |

## 功能边界

### 做什么
- 在 K 线模式下，点击任意交易日 K 线柱触发分析
- 识别 12 种 K 线形态（4 组：反转看涨、反转看跌、持续形态、高级组合）
- 根据形态命中情况展示完整讲解信息
- 形态判断逻辑和讲解文案均内置于前端代码

### 不做什么
- 不调用 AI/LLM 进行分析
- 不修改后端 API
- 不在 K 线未激活时触发点击分析
- 不提供实时/自动形态扫描
- 不做形态的量化准确率回测

## 核心设计

### 数据流

```
用户点击 K 线柱
  → ECharts click 事件（seriesType === 'candlestick'）
  → 获取 dataIndex + 完整 OHLC records
  → 传入 pattern detection 函数
  → 逐形态判断，收集所有命中
  → 渲染弹出窗口展示
```

### 形态检测函数架构

每个形态一个检测函数，签名统一：
```js
function detectXXX(records, index) → { hit: bool, detail: {...} } | null
```

### 辅助函数

- `isDowntrend(records, index, lookback)` — 判断是否处于下跌趋势（近 N 日 MA20 下行或收盘价持续走低）
- `isUptrend(records, index, lookback)` — 判断是否处于上涨趋势
- `isSmallBody(record)` — 判断实体是否偏小（实体长度 < 平均实体 * 0.3）
- `isBigBody(record)` — 判断实体是否偏大（实体长度 > 平均实体 * 1.5）
- `bodyRatio(record)` — 实体 / 影线比例
- `avgBody(records, lookback)` — 近 N 日平均实体大小

### 12 种形态检测规则

#### 第一组：反转看涨（下跌底部）

| 形态 | 所需K线 | 核心判断条件 |
|------|---------|-------------|
| 锤子线 | 1根 | 下跌趋势中；小实体；下影线 ≥ 2倍实体；上影线极短 |
| 吞没阳线 | 2根 | 下跌趋势中；前阴后阳；阳线实体完全包住阴线实体 |
| 早晨之星 | 3根 | 下跌趋势中；大阴线→小实体→大阳线；阳线收过阴线实体一半 |

#### 第二组：反转看跌（上涨顶部）

| 形态 | 所需K线 | 核心判断条件 |
|------|---------|-------------|
| 上吊线 | 1根 | 上涨趋势中；小实体；下影线 ≥ 2倍实体；上影线极短 |
| 吞没阴线 | 2根 | 上涨趋势中；前阳后阴；阴线实体完全包住阳线实体 |
| 黄昏之星 | 3根 | 上涨趋势中；大阳线→小实体→大阴线；阴线收过阳线实体一半 |

#### 第三组：持续形态

| 形态 | 所需K线 | 核心判断条件 |
|------|---------|-------------|
| 上升三法 | 5根 | 上涨趋势中；大阳线→2-4根小阴线回调（不破首阳最低）→大阳线创新高 |
| 下降三法 | 5根 | 下跌趋势中；大阴线→2-4根小阳线反弹（不破首阴最高）→大阴线创新低 |
| 十字星 | 1根 | 开盘≈收盘；根据位置判断含义（顶部看跌/底部看涨/中段持续）+ 变体分类 |

#### 第四组：高级组合

| 形态 | 所需K线 | 核心判断条件 |
|------|---------|-------------|
| 岛型反转 | 多根 | 跳空缺口→横盘→反向跳空缺口；被两个相反方向缺口隔离 |
| 三只乌鸦 | 3根 | 上涨后；连续三根大阴线；每根开盘在前一根实体内；收盘接近当日最低 |
| 三白兵 | 3根 | 下跌后；连续三根大阳线；每根开盘在前一根实体内；收盘接近当日最高 |

## 实际检测参数（校准后）

经实盘数据测试，教科书示意阈值过于严格。以下为校准后的实际参数：

| 参数 | 最终值 | 说明 |
|------|--------|------|
| 十字星 body/range | ≤15% | 初始 5%，实盘中极少出现 |
| 小实体判定 | ≤55% avgBody | 初始 35% |
| 大实体判定 | ≥120% avgBody | 初始 150% |
| 锤子/上吊线下影线 | ≥1.5×实体 | 初始 2× |
| 锤子/上吊线上影线 | ≤20% 波幅 且 ≤实体 | 初始 10% |
| 趋势 lookback | 5 日（默认） | 初始 10 日 |
| 斜率计算 null 处理 | 跳过 null 逐点累加 | 初始含 null 导致 NaN |
| 三乌鸦/三白兵收盘极值 | ≤35% 波幅 | 初始 25% |
| 星星形态中间 K 线 | body/range < 15% | 初始 5% |
| 趋势 fallback | 首尾比较法 | 无（仅斜率） |

## 风险与注意事项

- **趋势判断简化**：使用 MA20 斜率和近期收盘价方向作为趋势判断依据，不是精确的趋势线分析
- **岛型反转检测困难**：跳空缺口在日线级别较少出现，且需要前后多日数据，命中率会很低
- **K 线未激活时不触发**：需要在 click handler 中检查 `groupState.candlestick` 状态
- **与其他点击交互无冲突**：ECharts 已有的 tooltip 和 markPoint 不受影响，click 事件独立注册
- **阈值敏感**：判断阈值经过标普 500 日线数据校准，其他市场或时间级别可能需微调
